Podemos adicionar audios no nosso site, usando a tag audio.
Podemos adicionar de duas formas, uma é usando a tag audio e dentro dela adicionando controls, src, preload e type.
E podemos também adicionar usando a tag audio e o source src no qual usamos também o type e o preload, além do controls

Exemplo:
<audio controls preload="metadata">
<source src="audio.mp3" type="audio/mp3">
</audio>

Ou também usando <audio src="audio.mp3" controls preload="metadata">

Videos

Podemos também adicionar videos no nosso site

É praticamente a mesma coisa que o audio, você coloca a tag e os sources para adicioná-los.

<video controls width="" poster="thumb.jpeg">
<source src="caminho" type="video/tipo">
<video>

IMPORTANTE * SOBRE VIDEOS!

Videos internos, aonde eu coloco eles com o arquivo hospedado no meu computador por exemplo, geram mais custos para o cliente (ou para mim se eu for o dono), então o mais recomendável é upar o video no youtube e incorporar o código dele no site, assim reduzindo muito os custos.

Degradê em CSS

Podemos também adicionar degradê nas nossas páginas usando CSS, com o background-image: linear-gradient(to (aqui colocamos a direção, top, right, left, bottom) e colocamos as cores que queremos usar, separado por virgula.
Exemplo: background-image: linear-gradient(to right, blue, red, violet);
Se quisermos escolher o grau também podemos, ao inves de colocar a direção usamos o ângulo e deg, exemplo: 45deg
Podemos também adicionar quanto da cor queremos que apareça, adicionando a % ao lado, exemplo no azul eu posso colocar 10% ao lado dele.

Exemplo de site

Podemos também adicionar vários elementos de CSS para deixar nosso site mais bonito e agradável, alguns exemplos abaixo:

width: 1500px;
background-color:crimson;
text-align: justify; /* Deixa o texto justificado */
border-radius: 10px; /* cria uma borda arredondada nos cantos da caixa */
padding: 5px; /* Ele aumenta a lateral da caixa para não encostar muito no texto */
box-shadow: 5px 5px 10px black; /* Da sombreamento dos lados, embaixo, o nivel e a cor */
margin: auto; /* Define a margem dos 4 lados, baixo, cima, esquerda, direita de uma só vez, usando o auto ele deixa centralizado, podemos definir valores também caso queiramos, em px */

Tipografia

Temos também a tipografia que é o estudo de como escrever as coisas no papel, nela temos o porquê da importância dos estilos de escrita e fontes, como cada letra tem seu estilo, por exemplo as letras com serifa e sem serifa e as monoespaçadas.
Temos também a altura X que determinará qual será a altura das letras minúsculas, Temos também a letra das maíusculas aonde podemos pegar por exemplo o A e o Q.
O corpo é o Ascendente + descendente + Am + Ax.
Temos também as familias tipográficas que vão do light até o extranegrito
